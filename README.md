This Tableau Project involves creating interactive dashboards to visualize sales data for a retail company. The project includes data cleaning, integration, and visualization to provide insights into sales performance, customer demographics, and product trends. Key features include dynamic filtering, time-series analysis, and geographical mapping. The dashboards are designed to be user-friendly and support decision-making by highlighting key metrics and trends. This repository contains the Tableau workbook, data sources, and documentation for replicating and understanding the analyses. The goal is to enable data-driven strategies and improve business outcomes through intuitive visual analytics.
